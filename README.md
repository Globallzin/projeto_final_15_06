### Como compilar?

Primeiro clone o projeto usando:
```bash
git clone https://github.com/Globallzin/ProjetoFinalAlgoritmos1
```

Depois, para compilar usando o GCC, faça:
```bash
gcc *.c src/*.c -Iinclude -o main.exe
```

Depois, é só executar o arquivo <b>main.exe</b>.
